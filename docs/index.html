Designed for Pro Workflows

📌

Floating Overlays

Notes float above fullscreen apps and spaces. Pin notes to stay Always-on-Top or set them to read-only lock.

👁️

Focus-Aware Auto-Hide

Notes automatically fade out when you focus other windows and reappear instantly when you hover, click, or type.

🎹

Shortcut Builder

Map your own custom modifier keys (⌘, ⌥, ⇧, ⌃) and characters to create global hotkeys for Note triggers.

📑

Notes Dashboard

Manage all notes in a dynamic dual-column dashboard. Filter notes instantly with built-in search indexing.
